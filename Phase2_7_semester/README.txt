To run the code :

Add Dependencies jar files as external libraries.
Put stopwords.txt and websties.txt in the path of the project.
Change the path of pages to be downloaded in "Worker.java".
Change password and username of  userName in "Dbhandler.java" and "IndexerDbhandler.java" to user name and password of mysql on your computer.

Run index.jsp, type in your search word, and Voila.
